# qiqi-acme (基于 acme.sh 的中文一键脚本)

### 本项目是基于官方著名的开源项目 [acmesh-official/acme.sh](https://github.com/acmesh-official/acme.sh) 进行的中文汉化与自动化交互改造版。

主要为了方便习惯使用中文界面的开发者和 VPS 用户，提供全中文、交互式的 SSL 证书申请体验。

-------------------------------------
### 一键安装脚本
```bash
wget -P /root -N https://raw.githubusercontent.com/qiqi-style/qiqi_acme/main/qiqi_acme.sh && chmod +x /root/qiqi_acme.sh && bash /root/qiqi_acme.sh
```
---------------------------------------

### 🌟 功能与特点：

1. **多网络环境支持**：支持纯 IPV4、纯 IPV6、双栈 VPS 运行。
2. **多模式智能申请**：支持 80 端口模式与 DNS API 模式，无缝支持单域名与泛域名证书的申请。
3. **免域名测试**：80端口模式下，支持利用 `nip.io` 后缀域名解析 IP 申请测试用域名证书。
4. **国内 DNS 服务商友好**：深度集成了针对 Cloudflare、腾讯云 DNSPod、阿里云 Aliyun 托管解析平台的 DNS API 一键式申请。
5. **智能管理**：支持一键查询、撤销并删除当前已申请的域名证书。
6. **灵活续期**：支持手动一键续期所有证书或者指定某个特定域名证书。
7. **证书按域名隔离存储**：不同于原版的混存，**申请的证书文件将自动独立保存在 `/root/qiqissl/你的域名/` 文件夹中**，完美支持多域名多证书管理，便于给各类代理脚本或 Nginx 共用。
8. **自动防呆**：智能释放被占用的 80 端口，智能校对域名解析与当前机器 IP 是否匹配。

---------------------------------------
### ⚠️ 注意事项：
* 使用 80 端口模式时，会强制释放本机的 80 端口。如果您本机正在使用 Nginx、Caddy 等带自动申请证书功能的服务，请选择 DNS API 模式。
* 本项目纯为技术分享与中文交流使用。
